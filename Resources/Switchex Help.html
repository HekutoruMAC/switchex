Introduction

Thanks for using Switchex - WoW Realm Switcher!

Realm switching is made easy with Switchex. No need to edit your realmlist file manually and no programs that don't work correctly—just open Switchex and with the click of a button, you can be connected to your favorite server!

Features:

  • Add, edit, and delete servers
  • Create profiles for all your World of Warcraft installations
  • Include server names, IPs, websites, patch numbers, ratings, and notes
  • Open server websites and WoW from the main window
  • Check if the servers are online
  • Visit top server list websites
  • Visit WoW guides and wikis
  • Integrated "Check for Updates" feature downloads files straight from the GitHub project
  • Support for 32-bit and 64-bit WoW installations
  • Backup and restore your server database and realmlist.wtf file
  • Addon browser that allows you to manage your WoW addons

Main Window

The main window is where you can control the main functions of Switchex and the server list.

You can click the blue web link in the "Website" column to open that server's website.

Columns:

  • Online: Shows whether the server is online or not. Is checked if online and unchecked if offline. Can be accessed by clicking the Check Online button.

    Servers can be behind a firewall and will not show as online. You can always open the server website to see if it's still in business.

  • Name: The specified name of the server.
  • IP: The IP of the server, used to connect to it.
  • Website: The website of the server.
  • Patch: The World of Warcraft patch the server uses.
  • Rating: The rating you give the server.
  • Notes: The notes you give the server.
  • Profile: The profile that is currently assigned to the server.

Buttons:

  • Set Server: Sets the server in your realmlist.wtf file.

    If your WoW installation is in a folder like "Program Files", Switchex will ask for permission to write to the folder. Don't be alarmed if you see a User Account Control popup for "fileactions.exe".

  • Add Server: Adds a server to the server list.
  • Edit Server: Edits the selected server.
  • Delete Server: Deletes the selected server with confirmation.
  • Open WoW: Opens World of Warcraft.
  • Check Online: Checks if the servers are online.
  • Addons: Open the Addons browser.
  • Exit: Exits Switchex.

The Set Server, Open WoW, and Addons buttons will be unavailable if you are viewing all profiles at once.

Menus:

  • Server Lists: Top server list websites.
  • Guides and Wikis: Guides and wikis for WoW.

Options Window

The Options window can be accessed by opening Switchex and going to File > Options or by pressing Ctrl+O. It includes all of Switchex's options.

General

Check for updates at startup

If you check this box, Switchex will check for updates from the GitHub project while the program is loading. If there are updates, it will display a popup asking if you want to download them.

Open WoW after setting server

Checking this option, Switchex will automatically open World of Warcraft after setting the server. If turned off, Switchex will display a success notification after setting the server.

Delete servers with their profile

When this option is checked, servers will be deleted when the profile they are connected to is deleted. When not checked, they will be moved to the default profile.

A default profile can be selected from the Manage Profiles window.

Miscellaneous

The Empty Server List button will clear your server list, deleting all the servers and their information. Emptying your server list will create a backup if one does not exist. The Reset Settings button will reset the program settings to default and restart Switchex.

Backup

There's always the chance that your server list or realmlist.wtf files can become corrupted when edited, and that's what these buttons are for. Backing up a file will create a copy of it with the file extension .bak.

It's recommended that you make a backup of your server list before you install a new version of Switchex!

Restore

Switchex also has a restore option alongside the backup option. A backup (with the file extension .bak) must exist before a file can be restored.

If your WoW installation is in a folder like "Program Files", Switchex will ask for permission to write to the folder. Don't be alarmed if you see a User Account Control popup for "fileactions.exe".

Profiles Window

You'll find a new window under File > Manage Profiles.... Profiles are a new feature in version 7, allowing you to manage multiple WoW installations, all with their own servers! So you can have a profile for your Burning Crusade installation and your Wrath of the Lich King installation and keep their servers separate.

Click the Add New... button to add a new profile (up to a maximum of 10). Click the Delete button to delete a profile.

Selecting a profile using the dropdown will let you change its information. Once you make changes, click the Save button to save them (but don't worry—if you have unsaved changes, it will ask before closing the window). You can also click the Clear Changes button to clear the changes you've made to the profile before committing them.

There are also a couple of benefits to having a default profile: it will be the automatically-selected profile when adding new servers from the All profiles view and it will move servers from deleted profiles into the default profile.

Statusbar

In the statusbar on the bottom of the Switchex main window, you'll also notice that there are a few displays:

Profile

Profiles can be changed on the main window in the statusbar. Click the box next to Profile and select the profile you want.

"All" is a special option that shows you all servers from all profiles at once. It will disable the Set Server, Open WoW, and Addons buttons and the Open Game Folder, Open Addons Folder, and Open Realmlist.wtf options in the File menu.

Location

This will display the location of your current profile. So, for example, if you've set the location of your profile to C:\Program Files\World of Warcraft, it will show Location: C:\Program Files\World of Warcraft.

Patch

The patch display will show what version of World of Warcraft is installed with your current profile.

Server

The server display will show what server you currently have selected with your current profile.

Error Messages

Occasionally, you will receive an error message. I do my best to test the program, but I can't catch everything. You can help me! Error messages that are not supposed to happen at all will pop up in a box where you can copy them to your clipboard or send them straight to me via the contact form on my website.

If you send me the error messages you receive and what you were doing at the time, it will really help work the bugs out!
